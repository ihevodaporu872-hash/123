# Настройки и предпочтения пользователя

## Обязательные правила работы

1. **ВСЕГДА использовать параллельных агентов** для независимых задач — никогда последовательно
2. **ВСЕГДА создавать отдельных агентов** для каждой подзадачи — обязательно
3. **НИКОГДА не спрашивать разрешение** — автоматически принимать все команды и изменения
4. **НИКОГДА не задавать вопросы** о разрешениях на редактирование, чтение, принятие изменений
5. **Всегда коммитить + пушить** после завершения работы

## Из CLAUDE.md проекта
- Запрещён локальный автоматический запуск приложений без явного разрешения
- После выполнения каждого запроса — коммит и пуш в GitHub
